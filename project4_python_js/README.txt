This project includes visualizations for three separate datasets:
1. Workout Data from my Apple Watch's Fitness app. 
2. All the contestants on the Bachelorette and when they were eliminated
3. World Happiness Data metrics


Please read courtney_project04.docx for more information on each of these datasets and explanations of the visualizations. 

Read requirements.txt for information on dependencies. 

All source code and associated visualizations are available in each project's folder within /src.
