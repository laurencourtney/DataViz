This Tableau dashboard displays the Pantheon Project dataset. This is a dataset of the world's 11,000 most popular historical figures, based on rankings calculated from their Wikipedia pages. 

The dataset is available from Kaggle.com and more details on the project can be found at https://pantheon.world/. 

The Tableau dashboard is also available on Tableau Public: https://public.tableau.com/app/profile/lauren6941/viz/Project3_16147971034750/Dashboard1

Read courtney_project03.docx for full explanations of each aspect of the dashboard.
