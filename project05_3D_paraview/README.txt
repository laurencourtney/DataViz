This directory contains 6 total Paraview renderings of 2 separate 3D datasets, 3 renderings of each. 
The first dataset is a 3D image of a lobster encased in resin. 
The second dataset is a 3D image of a human skull.

The Paraview files all have the extension .pvsm, while screenshots of each of the visualizations are also present. 

For a full write up, please read courtney_project05.docx.
